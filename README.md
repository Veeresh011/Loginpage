# Login Page 🔐

A simple PHP login and registration system built with MySQL.

## Features
- User Registration
- User Login
- MySQL Database Connection

## Tech Stack
- PHP
- MySQL
- XAMPP (Apache)
- HTML/CSS

## How to Run
1. Install XAMPP
2. Clone this repo into htdocs folder
3. Start Apache and MySQL in XAMPP
4. Open phpMyAdmin and create database
5. Visit localhost/your-folder-name/login.php
